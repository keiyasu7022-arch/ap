# Rooret 🎡

JavaFXで作成したルーレットアプリです。

## 概要

高専2年生の授業の一環として開発しました。
通常のルーレットに「速度変更」「結果表示の切り替え」などの機能を追加し、より楽しく使えるよう工夫しています。

## 機能

- ルーレットの回転・停止
- 回転速度の変更（速く・遅く調整可能）
- 結果表示のON/OFF切り替え（右上のボタン）

## 使い方

1. 「回す」ボタンを押すとルーレットが回転します
2. 赤いボタンを押すとルーレットが停止します
3. 右上のボタンで結果表示の表示/非表示を切り替えられます

## 実行環境

- Java 11以上
- JavaFX

## セットアップ

> ⚠️ JDKとJavaFXが分離されているため、別途JavaFXのインストールが必要です。

1. [JavaFX](https://openjfx.io/) をダウンロード・インストール
2. 以下のコマンドで実行

```bash
java --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml -jar Rooret.jar
```

## 技術スタック

- Java
- JavaFX



# 単語帳アプリ 📖

PythonのTkinterで作成したシンプルな単語帳アプリです。

## 概要

独学でPython・Tkinterを習得し開発しました。
ファイル操作を使ってデータを永続化しており、登録・削除・一覧表示ができます。

## 機能

- 単語と意味の登録
- 単語の削除
- 登録済み単語の一覧表示
- 入力バリデーション（単語・意味が未入力の場合は登録・削除不可）
- 重複登録の防止（単語と意味が完全一致する場合のみ削除）

## 使い方

1. 単語・意味を入力して「記録」ボタンで登録
2. 「一覧表示」ボタンで登録済み単語を確認
3. 削除したい単語・意味を入力して「削除」ボタンで削除

## 実行環境

- Python 3.x
- Tkinter（Python標準ライブラリ）

## セットアップ

```bash
git clone https://github.com/keiyasu7022-arch/ap.git
cd ap
python 単語帳.py
```

## 技術スタック

- Python
- Tkinter
- ファイル操作（テキストファイルでデータ管理）
