Rooret

高専2年生の時に授業の一環としてjavafxで作成

使い方

・回すを押すとルーレットが回転して赤いボタンを押すとルーレットが止まる

工夫した点

・普通のルーレットでは面白くないので回転する速度を速くしたり遅くしたりできるようにした

・結果表示されるのがうざいと思う人もいるので右上のボタンを押したら表示と非表示を入れ替えれるようになっている

単語帳

独学でpythonのTkinterを使用し作成

使い方

・普通の単語帳と変わらないシンプルなものです

工夫した点

・一覧表示のボタンを押すと記録した単語の一覧が見れるようにした

・あえてファイル操作を使った

・単語、意味に入力していない場合、記録と削除がされない

・一緒の単語を登録する人いないと思うので意味と単語が一致した場合にのみ登録してあるのを削除
